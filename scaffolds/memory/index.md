# Memory Vault

Project-local agent memory. Populated by `/memory`, `/context`, `/reflect` skills.

## Directories

- `sessions/` — Per-issue session logs
- `context/` — Project context notes (overview, tech stack)
- `patterns/` — Reusable code patterns
- `entities/` — Domain entity documentation
- `templates/` — Templates for sessions, patterns, entities
- `prompts/` — Reusable prompts grouped by domain and workflow
- `team-configs/` — Learned team compositions from post-dev review
